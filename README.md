# Voice Calci 🎤🧮

Voice Calci is a voice-controlled scientific calculator built using Python and PyQt6. It supports both traditional button-based calculations and natural language voice commands, allowing users to perform calculations hands-free.

## Features

### Calculator Features

* Basic arithmetic operations (+, -, ×, ÷)
* Parentheses support
* Percentage calculations
* Powers and exponents
* Scientific functions:

  * Sine (sin)
  * Cosine (cos)
  * Tangent (tan)
  * Logarithm (log)
  * Natural Logarithm (ln)
  * Square Root
  * Factorial
* Mathematical constants:

  * π (Pi)
  * e (Euler's Number)

### Voice Features

* Speech recognition for calculations
* Natural language command support
* Text-to-speech answers
* Voice ON/OFF toggle
* Microphone button for hands-free input

### Additional Features

* Calculation history
* Persistent history storage
* Answer memory (Ans)
* Keyboard shortcuts
* Modern dark-themed interface
* Standalone Windows executable

## Example Voice Commands

```text
five plus six
seven times three
sin thirty
cos sixty
tan forty five
log one hundred
square root of one hundred forty four
answer plus five
double the answer
half the answer
square the answer
```

## Technologies Used

* Python
* PyQt6
* SpeechRecognition
* pyttsx3
* SymPy
* num2words
* PyInstaller

## Installation

### Option 1: Run the Executable

Download and run:

```text
VoiceCalc.exe
```

### Option 2: Run from Source

```bash
pip install PyQt6 SpeechRecognition pyttsx3 sympy num2words

python main.py
```
